# Contributing to Gita

Thanks for contributing.

## Prerequisites

- Node.js 22+
- pnpm 10+
- macOS, Linux, or Windows

## Local Development

```bash
pnpm install
pnpm dev
```

## Before Opening a PR

```bash
pnpm typecheck
pnpm lint
```

Please keep pull requests focused and include:
- A short summary of the change
- Screenshots/videos for UI updates
- Notes about testing done

## Code Style

- Use TypeScript consistently.
- Reuse existing components and patterns where possible.
- Keep UI behavior accessible and keyboard-friendly.

## Issues and Feature Requests

- Open an issue with clear reproduction steps for bugs.
- For features, describe the use case and expected behavior.

## Making Commits

- Use better-commits consistently.
