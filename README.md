# Gita - ဂီတ

Open-source music player with:
- Discord Rich Presence integration
- Search and streaming from YouTube and SoundCloud
- Local audio import and playback
- Queue controls, playlists, favourites, and persistence

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Build

```bash
# Windows
pnpm build:win

# macOS
pnpm build:mac

# Linux
pnpm build:linux
```
